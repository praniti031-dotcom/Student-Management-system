# Student-Management-system
📚 Student Task Management System
📌 Overview
The Student Task Management System is a web-based productivity application developed using HTML, CSS, and JavaScript. It is designed to help students organize their daily study routine, track their learning progress, and manage academic tasks efficiently.
The application allows users to log their study sessions, monitor focus hours, maintain a task calendar, and receive reminders for upcoming tasks, making it easier to stay consistent and productive.
✨ Features
📖 Study Tracker
Record the subject studied.
Select whether the session was Reading or Practice.
Enter the number of focused study hours.
📊 Performance Analysis
Analyzes study data based on the logged sessions.
Provides insights into study habits and productivity.
📅 Interactive Calendar
Add daily tasks directly to the calendar.
View scheduled tasks for each day.
✅ Task Completion
Mark tasks as completed using a checkbox.
Easily track pending and completed tasks.
⏰ Task Reminders
Receive reminders for scheduled tasks to stay on track.
💻 Simple & User-Friendly Interface
Clean and responsive design for a smooth user experience.
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive design
JavaScript (ES6) – Application logic, task management, reminders, and performance analysis
